# generator-fu-rnb

A [Yeoman](http://yeoman.io) generator for scaffold out a react-native app

## Installation

```bash
$ npm install -g yo
$ npm install -g git+ssh://git@github.com:mybigday/generator-fu-rnb.git
```

__*Note:*__ requires a node@^6.

## Usage

```bash
$ mkdir <app-name> && cd <app-name> && yo fu-rnb
```
