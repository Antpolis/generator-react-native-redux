# generator-fu-rnb

A [Yeoman](http://yeoman.io) generator for scaffold out a react-native app

## Installation

```bash
$ npm install -g yo
$ npm install -g @fugood/generator-fu-rnb
```

__*Note:*__ requires a node@^6.

## Usage

```bash
$ mkdir <app-name> && cd <app-name> && yo @fugood/fu-rnb
```
